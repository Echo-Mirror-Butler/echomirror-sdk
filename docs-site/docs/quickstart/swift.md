# Swift Quickstart

:::caution Coming soon
`EchoMirrorSDK` for iOS/macOS is not yet available. It will wrap [`echomirror-ffi`](pathname:///api/rust/echomirror_ffi/index.html), the same C-ABI shared library used by the Flutter SDK, and ship via Swift Package Manager.
:::

Track progress on the Swift package in the [EchoMirror SDK issue tracker](https://github.com/karanjadavi/echomirror-sdk/issues). In the meantime:

- [Core Concepts](../core-concepts) explains the architecture the Swift SDK will slot into
- The [Flutter Quickstart](./flutter) shows the same FFI pattern (`dart:ffi`) that Swift will use (Swift FFI) against the same shared library
