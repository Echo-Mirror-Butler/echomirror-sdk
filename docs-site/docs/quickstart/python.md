# Python Quickstart

:::caution Coming soon
Python bindings are not yet available. They will wrap [`echomirror-ffi`](pathname:///api/rust/echomirror_ffi/index.html), the same C-ABI shared library used by the Flutter SDK, via `ctypes`/`cffi`.
:::

Track progress on the Python bindings in the [EchoMirror SDK issue tracker](https://github.com/karanjadavi/echomirror-sdk/issues). In the meantime:

- [Core Concepts](../core-concepts) explains the architecture the Python bindings will slot into
- The [Rust Quickstart](./rust) and [Rust API Reference](pathname:///api/rust/) cover the underlying `echomirror-core`/`echomirror-stellar` logic that Python will wrap
